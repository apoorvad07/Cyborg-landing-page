# 🚀 TechFest – Cyborg Themed Landing Page

A modern, responsive landing page designed for a futuristic **TechFest IIT Bombay** event. The website features a cyberpunk-inspired UI with neon effects, interactive sections, and a clean user experience.

## 🌟 Features

- 🤖 Futuristic Cyborg-inspired design
- 📱 Fully Responsive Layout
- 🎨 Modern UI/UX with Neon Glow Effects
- 🏠 Hero Section with Call-to-Action
- ℹ️ About Section
- 🎯 Flagship Events Section
- 🖼️ Gallery Section
- 👥 Team Section
- 📞 Contact Section
- ⬆️ Smooth Navigation with Scroll-to-Top Button
- ✨ Hover Animations and Interactive Elements

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts

1. Clone the repository

```bash

```

2. Open the project folder.

3. Launch `index.html` in your browser.

No additional installation or dependencies are required.
